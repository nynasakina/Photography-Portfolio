## Features

- About Me 
- Gallery 
- FAQ
- Contact

## Tech Stack 
This project was built using the following:

- ReactJS
- Styled using CSS 
- [EmailJs](https://www.emailjs.com/) to allow emails to be sent directly to inbox
- [Firebase Google](https://firebase.google.com/) to store images
